# SQL Football Stats (Chelsea FC)

A simple SQL project analysing football match results using sample data from Chelsea FC.

The project includes:

- Table creation
- Sample dataset (8 matches)
- SQL queries for analysis
- SUM, AVG, ORDER BY, LIMIT examples

## How to Use

1. Open the `football_stats.sql` file in any SQL environment (SQLite, MySQL, PostgreSQL, DBeaver, etc.)
2. Run the CREATE TABLE and INSERT statements
3. Execute the queries to analyse the data

## Example Queries

- Total goals scored
- Average goals per match
- Best offensive match
- Worst defensive match
- Goal difference per match
